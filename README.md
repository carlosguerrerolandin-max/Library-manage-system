# Biblioshare: Sistema de Gestión de Bibliotecas Comunitarias

## Integrantes
* Carlos Alejandro Guerrero Landin

## Objetivo del Sistema
Optimizar el acceso a la lectura y la educación en entornos comunitarios mediante una herramienta informática ligera y de bajo coste, que permita inventariar, buscar y gestionar libros donados de forma eficiente sin necesidad de conexión a internet.

## Descripción de Funcionalidades
1. **Registro de Libros (Add Book):** Permite almacenar el título, autor y año de publicación de los libros.
2. **Visualización del Catálogo (Show Books):** Muestra el listado completo de los textos disponibles.
3. **Búsqueda Avanzada (Search Book):** Localiza un texto específico mediante su título (no distingue mayúsculas/minúsculas).
4. **Eliminación de Inventario (Delete Book):** Retira libros del catálogo cuando ya no estén disponibles.
5. **Persistencia de Datos:** Los datos se guardan automáticamente en un archivo local (`biblioteca.json`), evitando la pérdida de información al cerrar el programa.
